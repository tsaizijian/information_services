## 一、計畫名稱

醫療照護紀錄與異常事件分類（情緒、解便、癲癇、生理期）ME 計畫

## 二、計畫目的

1. 建立統一的醫療照護紀錄流程，減少遺漏與書寫差異。
2. 針對情緒異常、排便情形、癲癇發作、生理期等重要健康指標進行系統化追蹤。
3. 透過資料累積，協助醫師、護理人員及照服員掌握個案健康變化，作為醫療評估與照護調整的依據。
4. 建立異常事件之通報與緊急處置流程，確保個案安全。

## 三、適用對象與範圍

1. 對象：本機構（班級）內所有服務使用者。
2. 範圍：

   - 情緒與行為異常紀錄
   - 解便（排便）紀錄
   - 癲癇（或類似痙攣發作）紀錄
   - 生理期紀錄（限女性個案）

## 四、分類與紀錄內容

### （一）情緒／行為異常紀錄

**使用表單：新增情緒紀錄**

1. 基本欄位：姓名、發生日期時間、發生情境（如：用餐時、課堂中）、天氣、持續時間。
2. 引起異常因素：環境吵雜、與同儕衝突、被要求不喜歡的活動、不明原因…等。
3. 異常行為症狀：尖叫、哭、笑、吐口水、自傷（撞頭、打臉）、攻擊他人、脫衣、跑走、翻桌等。
4. 處理過程：安撫、隔離、適當約束、陪伴、維持環境安寧、檢查有無外傷等。
5. 是否急診／送醫：有嚴重外傷出血、呼吸困難、行為無法控制時需勾選並記錄。

> **ME 目標範例：**
>
> - 目標：個案每週情緒爆發次數較上月減少 20%。
> - 介入：調整環境刺激、預告活動、使用情緒圖卡、教導放鬆技巧。
> - 評值：每月底統計表單次數與內容，與團隊討論是否達標。

---

### （二）解便（排便）紀錄

**使用表單：新增解便紀錄**

1. 基本欄位：姓名、發生日期時間、距離上次解便已過時間。
2. 解便量：少量、中等、大量。
3. 顏色：正常、偏深、偏淺、帶血等。
4. 特殊狀況：腹瀉、便秘、硬塊、黏液、惡臭明顯…
5. 伴隨症狀：腹痛、噁心、嘔吐、發燒等。
6. 備註：是否使用軟便劑、灌腸、醫師指示等。

> **ME 目標範例：**
>
> - 目標：個案每 1–2 天能規律解便一次，避免便秘超過 3 天。
> - 介入：調整飲水量、纖維攝取、運動安排、必要時依醫囑用藥。
> - 評值：每週檢視解便紀錄，若超過 3 天未解便需通報護理／醫師。

---

### （三）癲癇發作紀錄

**使用表單：新增癲癇紀錄**

1. 基本欄位：發作日期時間、發作情境、天氣氣溫、持續時間。
2. 發作時症狀：

   - 意識變化：清楚、模糊、喪失意識。
   - 肢體症狀：雙眼上吊、四肢抽搐、口吐白沫、眼神呆滯等。
   - 其他：失禁、叫喊、嘴唇咬傷等。

3. 處理過程：

   - 保護頭部、移開危險物、鬆開衣物、協助側臥、維持呼吸道通暢、觀察時間。
   - 發作結束後給予保暖與休息，檢查有無外傷。

4. 是否急診／送醫：

   - 發作超過 5–10 分鐘未緩解
   - 連續發作
   - 呼吸困難、外傷嚴重或意識無法恢復等情形需立即送醫並記錄。

> **ME 目標範例：**
>
> - 目標：完整紀錄每次發作型態與時間，作為醫師調整藥物依據。
> - 介入：落實按時服藥、避免誘發因素（熬夜、強光刺激等）、建立緊急處置 SOP。
> - 評值：每次門診前彙整發作紀錄，與醫師討論。

---

### （四）生理期紀錄

**使用表單：新增生理期紀錄**

1. 基本欄位：姓名、出生日期、年齡、生理期開始日期。
2. 若系統有選項，可再記錄：經期天數、經血量（少/中/多）、腹痛程度、是否影響作息。
3. 備註：使用止痛藥情形、情緒變化、是否合併頭暈、嘔吐等。

> **ME 目標範例：**
>
> - 目標：掌握個案月經週期與經量變化，及早發現不規則出血或貧血徵象。
> - 介入：必要時安排婦科檢查、衛教個人衛生與經期情緒調適。
> - 評值：每 3–6 個月檢視紀錄是否出現週期過短／過長、經量過多等異常。

## 五、執行流程

1. **紀錄責任人**：

   - 日常照護由班導師／照服員填寫；
   - 護理人員定期查核並補充醫療相關欄位。

2. **紀錄時間點：**

   - 異常情緒／癲癇：事件發生後立即或事後 30 分鐘內完成。
   - 解便：每日固定時間檢查並登錄。
   - 生理期：第一天開始時登錄，結束後補上天數與異常情形。

3. **資料彙整：**

   - 每週由負責人列印或匯出紀錄，供班會／醫療會議討論。
   - 每月或每季進行統計分析（發作次數、便秘天數、情緒事件次數等）。

---

## 六、資料管理與隱私

1. 系統帳號權限分級，僅限相關照護人員查閱與編輯。
2. 若需提供資料給醫師或外部單位，須去識別化並取得家屬同意。
3. 定期備份資料，避免遺失。

---

## 七、預期效益

1. 讓照顧團隊能快速了解個案近期的情緒、腸胃、神經與生理狀況。
2. 提升異常事件的應變速度與處置一致性。
3. 協助醫師作出更精準的診斷與用藥調整。
4. 累積長期資料，作為評估照護品質與研究之基礎。

# 🟣 **三、癲癇發作紀錄（表單欄位詳細版）**

### **📌 A. 基本資料**

| 欄位           | 說明                            |
| -------------- | ------------------------------- |
| 發作日期、時間 | 精準到分                        |
| 發作情境       | 用餐、運動、休息、課堂、行走中… |
| 天氣氣溫       | 若與發作頻率相關（悶熱或寒冷）  |
| 發作持續時間   | 寫實際秒數（建議照護員計時）    |

---

### **📌 B. 發作症狀（多選）**

**1. 意識狀態**

- 意識清楚
- 意識模糊
- 喪失意識
- 眼神呆滯不回應

**2. 肢體症狀**

- 雙眼上吊
- 四肢僵直
- 四肢抽搐
- 身體突然癱軟
- 手腳無力
- 顫抖、抽動
- 唇部顫動

**3. 口部症狀**

- 口吐白沫
- 咬到舌頭
- 嘴唇變紫

**4. 其他**

- 失禁（尿/便）
- 大喊叫聲
- 無法站穩
- 發作後混亂、想睡

---

### **📌 C. 處理過程**

- 保護頭部
- 移開危險物（桌角、硬物）
- 協助側臥
- 解開衣領、鬆開衣物
- 保持呼吸道暢通
- 觀察與計時
- 發作後協助保暖
- 發作後休息
- 檢查有無外傷
- 外傷包紮

---

### **📌 D. 是否送醫**

| 選項                     | 說明           |
| ------------------------ | -------------- |
| 超過 5–10 分鐘未停止     | 重點欄位       |
| 連續性發作（間隔數分鐘） | 危險           |
| 呼吸困難、發紫           | 缺氧           |
| 外傷嚴重                 | 出血、骨折疑慮 |
| 意識未恢復               | 超過 5–10 分鐘 |

---

### **📌 E. 發作後狀況（新增欄位建議）**

- 嗜睡
- 頭痛
- 想吐
- 虛弱
- 情緒低落
- 能否恢復上課／活動

---

### **📌 F. 備註**

描述完整事件、前兆（如暈眩、失神）、家屬是否通知。
